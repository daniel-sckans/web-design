The MIDTERM

Create a site on a subject of your own choosing.

Technical Parameters

There is no formal requirement to use any particular technique, and you may use any of the HTML and CSS tools that we have addressed so far (e.g. HTML links; pictures/images; CSS rules such as position: fixed, font-family, margins and padding, and so forth).

Design Parameters

The site that you build must be orchestrated with your content. In other words, the style of the site — its color, tone, rhythm, typography, responsiveness, layout, and so forth — must react (think of a chemical reaction) with the concept of your site. While that reaction must be orchestrated, it can be of any type (the design can add to the concept, undermine it, contrast with it, compound with it...).
