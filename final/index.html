The Final

Create a complete standalone website.

Parameters

The site should feel like a complete, fully-designed and fully-implemented (or fully-finished) website.
The content is up to you (so long as it is consistent with our class requirements for open and safe dialogue).

Evaluation Criteria

The coherence of the site; the way that the visual design, its content, and its interactive elements are orchestrated together as a whole.
The significance of the content presented on the site, as well as the generosity of the site in offering content to the viewer (does content appear to be missing or lacking?).
The formal qualities of the site; the quality of any written text and origin media (e.g. pictures, videos, audio, and so forth).
The clarity and consideration of the UI (user interface).
The elegance with which the code of the site is used (is the code not only functional, but well-crafted?).

Notes

To be considered for grading, the assignment must be live and working during final critiques.
