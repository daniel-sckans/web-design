Assignment Six

Create a site that displays an animated poem.

Parameters

Use CSS animations (using the @keyframes and animation properties).
It must be live and publicly accessible.
You may use any of the web programming and design techniques we have discussed to date (e.g. images, multiple pages, display types).

Evaluation Criteria

The content of the poem itself (i.e. the written content).
The way that the poem visually unfolds for the viewer, including timing, placement, color, and composition.
The other media used to supplement the content of the poem (such as images, multiple pages, and shapes).

Inspiration

Rafael Rozendaal
Kinetic Typography
