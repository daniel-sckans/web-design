Assignment: Style an Unstyled Site

Prerequisites

You should use VSCode to execute your design (see Our Development Environment for details, and you should use it to push your site to the internet (see Git Cheat Sheet).

Copy the HTML from our dummy site.

Parameters

On our class Github you will find the HTML skeleton for a website, but it is entirely lacking any style. Add a link a CSS stylesheet to redesign the page. Use the tools we've talked about in previous classes, such as colors and hex colors, margins, padding, fonts and typographic tools, and display and position types. If you feel stuck regarding CSS, see this series of CSS guides.

You may not change the HTML you are given (except to link to a stylesheet).

When finished, upload the finished site to your own Github.

Rationale

Keeping the HTML for the site unchanging is a way of zeroing in on some CSS problems and principles. Sometimes it makes sense to change both your HTML and CSS to redesign your site, and sometimes it's more effective to change only the CSS. In this case we'll take the latter approach.

It's also common for web designers and front-end web developers (those working with the appearance and interactivity of a website) to be given a contract to change an existing site's style using only CSS. This is common because it allows the client to maintain the original content of their site.
