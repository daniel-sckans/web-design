Assignment Five

Parameters

Create a site with at least five (5) different pages, all linked to each other. Each page should connect to Google Analytics. The actual content of each page should be simple (e.g. it can be an h1 and nothing else), but each page should link back to one central stylesheet.
Conduct a usability test: have a partner click around the site without any direction from you personally.
On Tuesday, arrive with a hand-drawn map of where the user went on your site. Include a brief description of why you think they went to one page over another.

Considerations

Are you able to make sure that viewers continually end up back at the same pages? Are you able to do that through visual design? How can you gauge the attention that some pages receive over others? What information can you find that we didn't talk about in class?
