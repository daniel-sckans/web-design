Git Cheat Sheet

Step One: Create a Repository on Github.com

Only necessary if you're starting from scratch. Go to your Github page >> Repositories >> New.

Step Two: Write the Code For Your Site

Use VS Code and Browsersync to preview the site.

Step Three: Push the Site to Github

When you're ready to make your work live, open the terminal in VSCode by pressing CTRL-J. Then write the following:


  • git init [**only necessary if you're starting from scratch**]
  • git add .
  • git commit -m "**ADD YOUR CUSTOM MESSAGE HERE**
  • git push -f **https://YOUR-NEW-GITHUB-REPO-URL-HERE** master

Step Four: Enable Your Github.com Site

Only necessary if you're starting from scratch. Go to your Github.com repository's settings and — about midway down the page — for Github pages select "master branch".
