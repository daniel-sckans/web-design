Assignment Eight

Create a live chat server client and review one article from the Nielson Norman Group website.

Parameters

Your chat web app should use JavaScript to both send and receive chat messages to other web apps.
Your page should display both the other users currently connected and their messages.
Additionally, review (in about two paragraphs) one article or video from https://www.nngroup.com/. The video should relate directly to best practices for designing for the web.

Evaluation Criteria

The design of the chat app (are messages clearly displayed as they come in? Is the use of the app intuitive?).
Varied use of appropriate CSS techniques (e.g. the grid, flex, fonts, colors).
The inclusion of other page elements such as a page description, title, hero images, and so forth.
The site should be live and fully functioning at your Github site.
The Nielson Norman Group article review will be evaluated according to its insight into a relevant user experience principle.

Inspiration

MDN Introduction to JavaScript
Eloquent JavaScript (a free online book)
